Dashboard — First-time Setup

This wizard connects your dashboard to your own Firebase project, applies your branding, and creates your admin account. Run it once, on your live (deployed) URL.

1 · Before you start

You need a free Firebase project. Do these in the Firebase console first:

Firebase console checklist
  1. Create a project (Add project → name it → you can disable Analytics).
  2. Build → Firestore Database → Create database → start in Production mode (you'll paste rules next), pick a region.
  3. Build → Authentication → Get started → Sign-in method → enable Anonymous.
  4. Authentication → Settings → Authorized domains → add the domain you'll host on (e.g. yourname.github.io).
  5. Firestore → Rules tab → paste the rules below and Publish.
  6. Project settings ⚙ → General → Your apps → </> Web → register an app and copy the firebaseConfig object (you'll paste it on the next step).

Last steps
  1. If you haven't already, make sure firebase-config.js in this folder holds your config. and place it next to index.html, then redeploy.
  2. Open index.html and sign in with your passkey.
  3. Enter Admin Mode (your password) to add sections, entries, team members, and quick links.
  4. Delete setup.html from your host when you're done (optional, tidy).
