VL Dashboard — Data Migration

Single document dashboards/main → per-entity collections (Phase B). Run as the admin, once.

Safe by design: this tool only reads dashboards/main and writes the new collections. It never deletes the source doc — that stays as your rollback. Run Preview first (no writes), eyeball the counts, then Commit. Re-running is safe (idempotent — deterministic doc IDs overwrite rather than duplicate).
